Basilisk Engine
Basilisk Engine is a 3D engine built from scratch in Python. Offers many features for real-time rendering, physics, and game design, as well as an easy to use editor for building scenes and importing assets. Built the rendering pipeline with a forward and batched focused paradigm allowing users to render millions of objects at once.
Neural Network
A lightweight neural network I created in Python using Numpy. Utilized knowledge of calculus to implement gradient backpropagation. Trained with the MNIST data set to recognize handwritten numbers with 95% accuracy.
Level Editor
An easy-to-use level editor made with Pygame for quickly building 2D game worlds. Includes tools for tile sheet loading, custom rule tiles, collisions, saving/loading and exporting levels, and various editing tools. Optimized for making Pygame games to be better than other level editors on the market. Advanced tile rendering allows for millions of real-time tiles on the screen at once and infinite level sizes.
Minecraft Python Edition
A Minecraft clone that I programmed from scratch in Python and GLSL. Features procedural world generation, mining and building mechanics, voxel lighting, and inventory and crafting systems. Powered by an efficient voxel meshing algorithm and renderer allowing for high FPS despite being made in Python.
As a Mere Means
My first released game made with Pygame in two weeks for the Pirate Software Game Jam. A fun 2D action platformer with punchy combat and interesting mechanics. Features my first full asset pack made with Aseprite. Placed in the top 10% of entries in Theme and Artistic Style.
Portfolio Website
The website you are on right now. A sleek and minimal design with a focus on compact presentation. Design inspired by my favorite code editor and theme. Built entirely from scratch in HTML and CSS. Responsive to various screen sizes.
